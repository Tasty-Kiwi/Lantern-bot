# Lantern Radio

A bot with advanced wiki embedding features and color role setting.

## config.json

```json
{
  "token": "",
  "applicationId": "",
  "guildIds": [""],
  "isGlobal": true,

  "webhookUri": "<webhook uri to notify about crash>"
}
```

## Running

```bash
npm install
npm run deploy

npm run start
# or
node index.js
```
